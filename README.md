Proyecto: Analisis y Prediccion de Ventas

Este repositorio contiene una aplicacion desarrollada en Python y Streamlit que permite:

Cargar un dataset de ventas

Limpiar y preparar los datos

Visualizar graficos y realizar analisis exploratorio

Entrenar modelos de Machine Learning

Evaluar metricas de desempeÃ±o

Realizar predicciones

Ejecucion del proyecto

Instalar dependencias
Ejecutar en una terminal el siguiente comando:
pip install -r requirements.txt

Ejecutar la aplicacion Streamlit
streamlit run app.py

Esto abrira la interfaz web donde podras:

Subir tu dataset

Ver estadisticas

Hacer analisis exploratorio (EDA)

Entrenar un modelo

Obtener metricas

Realizar predicciones

Archivos del repositorio:

app.py --> Codigo principal de la aplicacion
requirements.txt --> Librerias necesarias
README.md --> Documentacion del proyecto
